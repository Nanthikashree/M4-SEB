# M4-SEB
AIM:
Write a program in C to read any digit, display it in the word using the Switch case statement,if none of the case is matching print "invalid  digit"?
PROGRAM:
```
#include <stdio.h>
int main()
{
    int n;
    scanf("%d",&n);
    switch(n)
    {
        case 1:
        printf("One");
        break;
        case 2:
        printf("Two");
        break;
        case 3:
        printf("Three");
        break;
        case 4:
        printf("Four");
        break;
        case 5:
        printf("Five");
        break;
        case 6:
        printf("Six");
        break;
        case 7:
        printf("Seven");
        break;
        case 8:
        printf("Eight");
        break;
        case 9:
        printf("Nine");
        break;
        case 10:
        printf("Ten");
        break;
        default:
        printf("Invalid");
        break;
    }
}

```
